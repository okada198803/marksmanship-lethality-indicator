# MaLI (Marksmanship Lethality Indicator) ver47 - Tactical Eval

**MaLI** は、射撃訓練のパフォーマンスを科学的・多角的に測定するためのオープンソース・タクティカル・アナリティクス・ツールです。デバイスのマイクを用いた「精密ショットタイマー」と、画像解析による「着弾点評価」を統合し、データに基づいたフィードバックを即座に提供します。

---

## 🚀 主な機能

### 1. タクティカル・ショットタイマー (Audio Analysis)
* **高精度検知**: Web Audio APIを使用し、発射音をリアルタイム解析。
* **初弾・スプリット計測**: 初弾速度（First Shot）から各弾の間隔（Split Time）までを0.01秒単位で記録。
* **リロード検知**: 特定弾数でのリロード所要時間を自動計測し、操作習熟度を評価。

### 2. マルチターゲット画像解析 (Visual Analysis)
* **複数標的一括対応**: 射距離が異なる複数の標的写真を一度にアップロード・解析可能。
* **デュアルモード弾痕解析**: シールあり（ピクセル解析）と実弾痕（AI解析）を撮影状況に応じて切り替え可能。詳細は「11. デュアルモード弾痕解析」を参照。
* **個別散布界記録**: 各的ごとの散布界を独立してデータ化。

### 3. プロフェッショナル評価システム (A-D Grading)
* **目標値ベースの判定**: 設定した目標値を「Bランク（基準）」とし、実績に応じて **A (Elite)** から **D (Improvement Required)** までを判定。
* **4軸レーダーチャート**: 「精度」「速度」「制御」「暴露」のバランスを可視化。

### 4. 高度なデータエクスポート
* **Excel最適化CSV**: 射距離構成が日付に誤変換されるのを防ぐフォーマットを採用。
* **詳細統計記録**: 各弾のタイムスタンプ、距離ごとの散布界、各項目のグレードをすべて網羅。

## ✨ ver47 新機能・改善点

### 5. セッション履歴管理 (History)
* **ローカル保存**: 最大20セッションをブラウザ上（localStorage）に自動保存。インターネット接続不要。
* **履歴一覧表示**: 射手ID・射距離・日時・各軸グレードを一覧で確認可能。
* **個別削除**: 不要なセッションを任意に削除できる。

### 6. セッション比較機能 (Compare)
* **最大4セッション同時比較**: 初弾・平均間隔・散布界・命中数・総暴露時間・発射数を並べて比較。
* **最良値のハイライト**: 比較表内で最も優れた値を緑色で自動ハイライト。
* **タブ構成**: SESSION / HISTORY / COMPARE の3タブでUIを整理。

### 7. ショットタイミンググラフ (Shot Timing Graph)
* **インターバルバーグラフ**: 各ショット間のインターバルを棒グラフで可視化。
* **目標ラインの表示**: 設定した目標間隔（cfg-target-split）を基準線として重ね表示。
* **カラーコーディング**: 目標以内（緑）・超過（赤）・リロード（青）で色分け表示。

### 8. プリセット設定 (Preset)
* **至近距離プリセット**: CQB想定。5発・命中4・散布界40cm・初弾0.5s・間隔0.5s・リロード3.5s（3発目）。全射距離15m。
* **複数距離プリセット**: 多目標想定。10発・命中5・散布界80cm・初弾2.5s・間隔2.5s・リロード3.5s（6発目）。MAG-1前半300m→後半200m・MAG-2は100m。
* **カスタム**: 手動で各パラメータを自由設定。

### 9. 各ターゲット射距離の個別設定
* **Shot単位の射距離入力**: 発射数に応じてShot #1〜#Nの射距離を個別に入力できる行を自動生成。
* **MAGグループ表示**: リロード弾数が設定されている場合、MAG-1 / MAG-2 でグループ分けして表示。
* **CSV出力への反映**: 各ショットの射距離が個別にCSVへ記録される。

### 10. パフォーマンス改善
* **マイク検出精度向上**: `echoCancellation: false`・`noiseSuppression: false`・`autoGainControl: false` を明示的に設定し、発射音の特性に最適化。
* **RMS（二乗平均平方根）エネルギー検出**: 単純なピーク検出からRMSベースの検出へ変更し、誤検知を低減。
* **波形ビジュアライザーへの閾値ライン表示**: 設定した検出閾値をリアルタイムに波形上へ赤点線で描画。
* **画像解析の高速化**: サンプリング間隔を画像サイズに応じて動的に調整。大画像での処理速度を改善。
* **Canvas最適化**: `willReadFrequently: true` ヒントを指定し、ピクセル読み取りの繰り返しを高速化。
* **クラスタリング計算の最適化**: 弾痕クラスタリングの距離計算をルート不要の二乗比較へ変更し、近接弾痕の誤結合バグを修正。
* **ショット検出時のビジュアルフィードバック**: 検出直後にタイマー表示が黄色にフラッシュ。
* **マイクステータスインジケーター**: INACTIVE / ACTIVE を視覚的に表示するドットインジケーターを追加。
* **エラーハンドリング強化**: マイクアクセス拒否時に明示的なアラートを表示。
* **ショットログ表示**: タイマー下に各発のタイムと前発からのインターバルをリアルタイム表示。

### 11. デュアルモード弾痕解析 (Dual-Mode Visual Analysis) ⭐ NEW

実際の撮影状況に応じて2つのモードを切り替えて使用できます。

#### 🔵 シールありモード（ピクセル解析）
* **対象**: 弾痕に青色シールを貼ってから撮影した標的。
* **動作**: ピクセル単位で青色領域を検出・クラスタリングし、命中数と散布界を算出。
* **感度スライダー**: 1（厳格）〜5（緩い）の5段階で検出感度を調整可能。感度4以上では検出ピクセルを黄色点で可視化。
* **オフライン動作**: インターネット接続・APIキー不要。完全ローカル処理。

#### 🤖 AI解析モード（Gemini API）
* **対象**: 実弾による穴・裂け目・金属片が見える標的写真をそのままアップロード。シール不要。
* **動作**: Google Gemini 2.0 Flash APIが画像を直接解析し、弾痕の数・座標・散布界・所見をJSON形式で返答。結果を画像上にオーバーレイ表示。
* **無料で使用可能**: Gemini APIの無料枠（1日1,500リクエスト）で動作。GitHub Pages等の静的ホスティング上でも利用可能。
* **APIキー管理**: [Google AI Studio](https://aistudio.google.com/apikey) で取得したAPIキーをアプリ内に入力。キーはlocalStorageに保存され、HTMLには含まれないため安全。

> **Gemini APIキーの取得手順**
> 1. [Google AI Studio](https://aistudio.google.com/apikey) にアクセス
> 2. Googleアカウントでログイン
> 3. 「Create API Key」をクリック
> 4. 表示された `AIza...` から始まるキーをコピー
> 5. アプリの「🤖 AI解析」モードのAPIキー欄に貼り付け

## 🛠 使い方 (Usage)

1. **Setup**: `mali_v47.html` をブラウザで開きます。
2. **Preset / Scenario**: プリセット（至近距離 / 複数距離 / カスタム）を選択し、必要に応じてパラメータを調整します。各ショットの射距離を個別に設定します。
3. **Timer**: `START` を押し、ランダムなブザー音の後に射撃を開始します。
4. **Analysis**: 終了後、解析モード（シールあり / AI）を選択し、各ターゲットの写真をアップロードします。
5. **Report**: 表示されたチャートと統計を確認し、`セッション保存` で履歴に記録、`CSV出力` でファイルを保存します。
6. **Compare**: HISTORYタブで過去セッションを確認。COMPAREタブで複数セッションを並べて比較します。
