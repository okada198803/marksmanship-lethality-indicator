# 射撃術技量評価ツール MaLI ver60 - Marksmanship Lethality Indicator

**MaLI** は、射撃訓練のパフォーマンスを科学的・多角的に測定するためのオープンソース・タクティカル・アナリティクス・ツールです。デバイスのマイクを用いた「精密ショットタイマー」と、画像解析による「着弾点評価」を統合し、データに基づいたフィードバックを即座に提供します。

---

## 🚀 主な機能

### 1. タクティカル・ショットタイマー (Audio Analysis)
* **高精度検知**: Web Audio API を使用し、発射音をリアルタイム解析（RMSエネルギー検出）。
* **初弾・スプリット計測**: 初弾速度から各弾の間隔まで 0.01 秒単位で記録。
* **リロード検知**: 特定弾数でのリロード所要時間を自動計測。
* **検出閾値**: デフォルト 0.50（スライダーで 0.10〜0.99 の範囲で調整可能）。
* **不感時間**: デフォルトは実銃（0.07s）。ガスブローバック（0.15s）に切り替え可能。
* **リロード不感時間**: スライダーで 0.5〜5.0s の範囲で調整可能（デフォルト 2.0s）。リロード中の誤検知を防止。

### 2. マルチターゲット画像解析 (Visual Analysis)
* **複数標的対応**: 複数の標的写真をアップロード・解析可能。初回アップロード後に「＋ 写真を追加アップロード」ボタンが表示され、1 枚ずつ追加も可能（iOS 複数選択非対応時の回避策）。
* **デュアルモード弾痕解析**: 手動マーク（タッチ操作）とシールあり（ピクセル解析）を切り替え可能。
* **個別散布界記録**: 各的ごとの散布界を独立してデータ化。

### 3. プロフェッショナル評価システム (A-D Grading)
* **目標値ベースの判定**: 設定した目標値を基準に A〜D の 4 段階グレードを自動判定。
* **5 軸評価**: 速度・精度・制御・暴露・適応のうち、シナリオが評価する軸のみを動的に表示するレーダーチャート。

### 4. 4 タブ構成
* **SESSION**: 射撃・解析・個別成績の確認。
* **HISTORY**: 単一シナリオのトレンドグラフ＋セッション一覧。単体および複数選択した CSV 出力が可能。
* **ASSESSMENT**: 3 射撃要領の総合評価。5 つのレーダーチャートで多角的に分析。
* **COMPARE**: 複数セッションの数値比較＋仮想敵との Combat Simulation（3 プリセット）。

## 🎯 射撃要領プリセット

MaLI には 3 つの射撃要領プリセットが組み込まれています。

### 至近射撃（姿勢変換）
目標 1 に 4 発・目標 2 に 3 発・目標 3 に 3 発（計 10 発）、全て射距離 25m、7 発目射撃後に弾倉交換。

| 指標 | 評価内容 | 評価 B 基準 |
|---|---|---|
| 速度 | 平均射撃間隔・射撃合計時間 | 間隔 2.0s・合計 30s |
| 精度 | 命中数・射弾散布 | 各目標 2 発命中・散布界 20cm 以内 |
| 制御 | 弾倉交換所要時間 | 4.0s 以内 |
| 暴露 | 目標ごとの暴露有無（チェックリスト） | 暴露 1 回以下 |
| 適応 | 評価しない | — |

### 制圧致命射撃（昼間）
目標 1 に 5 発、射距離 10m、リロードなし。

| 指標 | 評価内容 | 評価 B 基準 |
|---|---|---|
| 速度 | 平均射撃間隔・射撃合計時間 | 間隔 0.8s・合計 5s |
| 精度 | 命中数・射弾散布 | 3 発命中・散布界 10cm 以内 |
| 制御 | 5 発目の頭部命中（チェックリスト） | 命中あり |
| 暴露 | 評価しない | — |
| 適応 | 評価しない | — |

### 制圧致命射撃（夜間）
目標 1 に 5 発、射距離 10m、リロードなし。

| 指標 | 評価内容 | 評価 B 基準 |
|---|---|---|
| 速度 | 平均射撃間隔・射撃合計時間 | 間隔 1.5s・合計 8s |
| 精度 | 命中数・射弾散布 | 3 発命中・散布界 15cm 以内 |
| 制御 | 5 発目の頭部命中（チェックリスト） | 命中あり |
| 暴露 | 評価しない | — |
| 適応 | 昼間結果からの変化（速度・精度・合計時間の均等平均） | 悪化率 20% 以内 |

## 📊 評価指標の詳細

### 5 つの評価軸

#### 速度 / Speed
射撃の素早さを評価します。

| 細部指標 | 計測内容 |
|---|---|
| 平均射撃間隔 | 純粋な連射間隔の平均。初弾・リロード区間・MAG2 初弾を除外して計算 |
| 射撃合計時間 | スタートから最終弾発射までの経過時間 |

#### 精度 / Accuracy
射撃の正確さを評価します。

| 細部指標 | 計測内容 |
|---|---|
| 命中数 | 手動マーク確定またはシール検出の合計命中数 |
| 最大散布界 | 複数標的のうち最大の散布界（cm） |

#### 制御 / Control
シナリオに応じて評価内容が変わります。

| シナリオ | 評価内容 |
|---|---|
| 至近射撃（姿勢変換） | 弾倉交換所要時間：`(MAG2 初弾時刻 − MAG1 最終弾時刻) − 目標初弾速度` |
| 制圧致命射撃（昼/夜） | 5 発目の頭部命中の有無（チェックリスト・A/D 判定） |

#### 暴露 / Exposure
射撃時の暴露管理を評価します（至近射撃のみ）。

目標ごとに「暴露あり/なし」をタップ入力するチェックリスト方式。暴露回数が目標値（1 回）以下で B 以上の評価となります。

#### 適応 / Adapt
昼間と夜間の間でのパフォーマンス維持能力を評価します（制圧致命射撃夜間のみ）。

射撃間隔・散布界・命中率・合計時間の 4 項目の昼夜間変化率を均等平均し、悪化率を算出します。

> **悪化率の計算例**: 射撃間隔が昼間 0.8s → 夜間 1.0s に増加した場合、変化率は +25%。4 項目の変化率を均等平均し、20% 以内で B 評価。

## 🏆 グレード（A〜D）の計算方式

### 個別指標のグレード判定

**「低い方が良い」指標（時間・散布界など）**

| グレード | 条件（実測値 ÷ 目標値） | 意味 |
|---|---|---|
| **A** | 0.85 以下 | 目標を 15% 以上上回る |
| **B** | 0.86〜1.15 | 目標達成・標準的な成績 |
| **C** | 1.16〜1.40 | 目標をやや下回る |
| **D** | 1.41 以上 | 目標を大幅に下回る |

**「高い方が良い」指標（命中数など）**

| グレード | 条件（実測値 ÷ 目標値） | 意味 |
|---|---|---|
| **A** | 1.10 以上 | 目標を 10% 以上上回る |
| **B** | 0.90〜1.09 | 目標達成 |
| **C** | 0.70〜0.89 | 目標をやや下回る |
| **D** | 0.69 以下 | 目標を大幅に下回る |

### レーダーチャート軸グレードの平均化

複数の細部指標がある軸は A=4 点・B=3 点・C=2 点・D=1 点に換算して平均し、再びグレードに変換します。

| 平均スコア | 軸グレード |
|---|---|
| 3.5 以上 | **A** |
| 2.5〜3.4 | **B** |
| 1.5〜2.4 | **C** |
| 1.4 以下 | **D** |

> 例：命中数 A（4 点）・散布界 C（2 点）の場合、精度軸 = 平均 3.0 点 → **B**

## 📈 ASSESSMENT タブ（3 射撃要領 総合評価）

3 つの射撃要領を実施・保存すると ASSESSMENT タブで総合評価を確認できます。

### 総合グレードの算出
評価した指標のみを均等荷重で平均して OVERALL グレードを算出します。未実施のシナリオ・未評価の軸は対象外です。

### 5 つのレーダーチャート

| チャート | 内容 |
|---|---|
| 総合 | 3 シナリオを速度・精度・制御・暴露・適応の 5 軸に統合 |
| 至近射撃（姿勢変換） | 当該シナリオの評価軸のみ表示 |
| 制圧致命射撃（昼間） | 当該シナリオの評価軸のみ表示 |
| 制圧致命射撃（夜間） | 当該シナリオの評価軸のみ表示 |
| 適応（昼夜比較） | 昼間と夜間の各指標を対比して表示 |

### 総合レーダーチャートの 5 軸統合ロジック

| 軸 | 統合方法 |
|---|---|
| 速度 | 3 シナリオの速度グレードを均等平均 |
| 精度 | 3 シナリオの精度グレードを均等平均 |
| 制御 | 3 シナリオの制御グレードを均等平均（弾倉交換・頭部命中を同一軸として統合） |
| 暴露 | 至近射撃（姿勢変換）のみ評価 |
| 適応 | 制圧致命射撃（夜間）の昼夜比較グレード |

## 📂 HISTORY タブ CSV 出力

セッション一覧からの CSV 出力が可能です。

* **単体出力**: 各セッションカードの「📊 CSV」ボタンで個別出力。
* **まとめて出力**: 各カードのチェックボックスで複数選択し、上部の「📊 選択したセッションをCSV出力」で一括出力。「全選択」「全解除」ボタンも利用可能。

CSV 出力列：Timestamp / ShooterID / ScenarioType / TotalShots / Hits / 速度・精度・制御・暴露・適応の値とグレード / 射距離 / 各弾タイム

## ⚔️ Combat Simulation（仮想敵との交戦シミュレーション）

モンテカルロ法（1,000 試行）で交戦勝率を算出します。

### 仮想敵プリセット（3 種・5 指標）

| 指標 | 軽武装ゲリラ | 正規部隊（標準） | エリート部隊 |
|---|---|---|---|
| 速度（射撃間隔） | 1.20s ±0.30 | 0.80s ±0.20 | 0.45s ±0.10 |
| 精度（命中率） | 55% ±15 | 72% ±10 | 90% ±5 |
| 制御 | 50% ±20 | 70% ±15 | 90% ±8 |
| 暴露管理 | 60% ±20 | 75% ±12 | 92% ±6 |
| 適応 | 65% ±15 | 75% ±12 | 88% ±8 |

### セッション・Assessment 結果との比較
自分のデータとして通常セッション（個別）または ASSESSMENT の総合評価結果を選択できます。セッションの種類（シナリオタイプ）を自動判定し、**利用可能な指標のみ**を比較対象として自動選択します。

| 選択データ | 使用指標 |
|---|---|
| 至近射撃（姿勢変換） | 速度・精度・制御・暴露 |
| 制圧致命射撃（昼間） | 速度・精度・制御 |
| 制圧致命射撃（夜間） | 速度・精度・制御・適応 |
| ASSESSMENT 総合結果 | 速度・精度・制御・暴露・適応（評価済み全指標） |

### 勝敗判定と結果表示
各指標を 0〜1 スケールで正規化し、利用可能な指標の均等平均スコアを比較。自分の実測値には ±12% の正規分布ノイズを付加した現実的な勝率を算出します。

| 勝率 | 評価 |
|---|---|
| 80% 以上 | 🟢 DOMINANT 圧倒的優位 |
| 65〜79% | 🟡 FAVORABLE 有利 |
| 50〜64% | 🟠 CONTESTED 拮抗 |
| 35〜49% | 🔴 DISADVANTAGED 不利 |
| 35% 未満 | 💀 CRITICAL 危険域 |

## 🛠 使い方 (Usage)

1. **Setup**: `mali_v60.html` をブラウザで開きます。
2. **Preset / Scenario**: プリセット（至近射撃・制圧致命昼間・制圧致命夜間・カスタム）を選択し、各ショットの射距離を個別に設定します。
3. **Timer**: 検出閾値（デフォルト 0.50）・不感時間（デフォルト 実銃 0.07s）・リロード不感時間を確認し、`START` を押してランダムなブザー音の後に射撃を開始します。
4. **Analysis**: 終了後、解析モード（手動マーク / シールあり）を選択し、各ターゲットの写真をアップロードします。
   - **手動マーク**: 拡大しながらタップで弾痕を指定 → 「✅ マーク確定・スコア計算」
   - **チェックリスト**: 制御（頭部命中）・暴露の入力欄が表示されたらタップで入力
5. **Report**: 「💾 セッション保存」で履歴に記録。「📊 CSV 出力」でファイルを保存。
6. **HISTORY**: 単一シナリオのトレンドグラフで成長の推移を確認。カード単体またはチェックボックスで複数選択して CSV 出力できます。
7. **ASSESSMENT**: 3 シナリオをすべて保存すると総合評価が表示されます。射手ボタンで切り替え、5 つのレーダーチャートで多角的に分析します。
8. **COMPARE**: セッションまたは ASSESSMENT 総合結果を選択し、3 種類の仮想敵との交戦勝率を算出します。

## ⚙️ 設定パラメータの意味と目標値の決め方

### 射手 ID
誰のデータかを識別するための任意の文字列。複数人使用時や後から見返す際に使います。

### 標的高さ (cm)
使用する標的紙の実際の縦寸法です（写真の縦枠全体ではなく、標的紙そのものの高さ）。標準的な標的は 100cm。散布界の算出に使用します。

### 目標値設定の目安

| パラメータ | 初心者 | 中級者 | 上級者 |
|---|---|---|---|
| 初弾速度 | 2.0〜3.0s | 1.0〜2.0s | 0.5〜1.0s |
| 連射間隔 | 1.5〜3.0s | 0.8〜1.5s | 0.3〜0.8s |
| 散布界（25m） | 40〜60cm | 20〜40cm | 10〜20cm |
| 弾倉交換 | 5.0〜8.0s | 3.0〜5.0s | 1.5〜3.0s |

### リロード所要時間の定義
`リロード所要時間 = (MAG2 初弾の時刻 − MAG1 最終弾の時刻) − 目標初弾速度`

リロード後に照準を合わせる時間（目標初弾速度）を差し引いた、純粋な銃の操作時間を評価します。

> 例：MAG1 最終弾 1.0s・MAG2 初弾 5.0s・目標初弾速度 2.0s → リロード所要時間 = **2.0s**

## ⚠️ 散布界計測に関する注意事項

散布界はピクセル比（標的高さ設定 ÷ 画像縦ピクセル数）で算出します。精度を確保するために以下を統一してください。

* **同一機種・同一カメラアプリ・同一アスペクト比**で撮影
* **メインカメラのみ使用**（超広角レンズは歪みが大きく不可）
* 標的を**画面中央・縦いっぱい**に収まるよう構図を合わせる
